##### Deskripsi:
SecurePass adalah aplikasi untuk menyimpan dan mengelola informasi login akun secara lokal dan aman. Data utama yang digunakan adalah data akun layanan, nama pengguna, dan kata sandi. Pengguna aplikasi adalah individu yang ingin mengamankan akses digital mereka.

##### Spesifikasi:
1. Pengguna dapat menambahkan, mengubah, dan menghapus data akun layanan.
2. Sistem dapat mencatat nama layanan, alamat email, kata sandi, dan tanggal pembaruan terakhir.
3. Pengguna dapat mencari data akun berdasarkan nama layanan menggunakan Sequential dan Binary Search.
4. Pengguna dapat mengurutkan data akun berdasarkan nama layanan secara alfabetis atau waktu input menggunakan Selection dan Insertion Sort.
5. Sistem dapat menampilkan statistik jumlah total akun yang tersimpan dan klasifikasi kekuatan kata sandi.
